# Sorting Visualizer

A production-ready, React + TypeScript sorting algorithm visualizer with Canvas rendering, step-by-step mode, and educational info panels.

## Tech Stack
- **React 18** + **TypeScript**
- **Zustand** (state management)
- **Canvas API** + `requestAnimationFrame` (rendering)
- **Tailwind CSS** (styling)
- **Vitest** (testing)
- **Vite** (bundler)

## Algorithms
Bubble Sort, Insertion Sort, Merge Sort, Quick Sort, Heap Sort

## Getting Started

```bash
npm install
npm run dev       # Development
npm run build     # Production build
npm test          # Unit tests
